Class RigidBody


Extends Observable.
A rigid body consists of a number of points in space. In this engine calculations on rigid bodies are done on the centre of mass. This greatly simplifies the mathematics involved in calculating the relative velocities of each point in space occupied by the rigid body.
Defined in: rigidBodyPhysics.js.
